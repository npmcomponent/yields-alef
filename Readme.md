# [alef](http://alef.hagilda.com/)

  [Alef](http://alef.hagilda.com/) is an open source bilingual web-font

## Installation
### Using [Component](http://component.io/):
    $ component install yields/alef
### Using [Bower](http://bower.io)
    $ bower install alef

## Example

```css
body {
  font-family: 'alef';
}
```
